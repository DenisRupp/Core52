Caching

Caching is controlled on a per-query basis and is disabled by default. To enable caching, pass TRUE as the second argument the execute() method on your DatabaseConnection instance:

$result_cached = database()->execute($query, TRUE);
$result_cached = Database::e($query, TRUE); // this also works

If you’re running using a DatabaseQuery:

$result_cached = $query->run(TRUE);

Note that contrary to the execute() method, run() enables caching by default.

If you’re running using Database modifiers, you can use Database::cache($setting) to enable/disable caching.

Caching is provided by the DatabaseCache class. For more details, see the Database page in “Core52 Libraries”.
