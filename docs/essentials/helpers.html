Helpers

Usage

Helpers are collections of related functions that do specific tasks, like formatting, filtering, etc.

Store Helpers in your app/helpers directory.

Loading

Helpers must be loaded manually. You can load helpers using the core_load_helper() function:

core_load_helper('my_helper_name');  // leave off the .php
core_load_helper('format', TRUE);  // load a core helper

Core Helpers

Core52 includes an array of useful helpers that you can use in your application:

  • Composing raw or bulk email messages
  • Rendering HTML form fields (borrowed from CodeIgniter)
  • Filtering and sanitizing input
  • Formatting JSON objects
  • Deep-cloning objects
  • Formatting dates and times
  • Inflecting words

See the Core52 Helpers section in the documentation for more details.
