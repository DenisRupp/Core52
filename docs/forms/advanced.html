Advanced Form Processing

Processing Individual Fields

validate_field($field, $run_filters)

Validate a single field. Returns TRUE if successful (note that empty fields are considered valid by default unless the required rule is specified), FALSE if not.

render_field($field, $forced_value)

Renders a single field. Returns an HTML string. You can force the value of the field by passing a second argument.

field_name($field)

Returns the human-friendly field name (uses the label setting if present).

get_file($field)

Returns the $_FILES array for a file upload field.

Changing Field Settings

The following methods are available for changing field settings on-the-fly in your controller logic.

add_filter($field, $filter), add_filters(array $filters)

Sets the filter setting for a field or multiple fields.

add_rule($field, $rule), add_rules(array $rules)

Appends validation rules to a field or multiple fields.

set_attributes($field, $options), set_attributes_multi($values)

Set the HTML attributes for a field or multiple fields.

set_default($field, $default_value), set_defaults(array $defaults)

Set the default values for a field or multiple fields.

set_field($field, $value)

Sets value of field. Overrides submitted form data. Used for replacing the data gathered from the $_POST[] array.Â  Can be helpful if wanting to change the value of a field before it gets returned from ->get_fields() or ->render().

set_label($field, $label), set_labels(array $labels)

Set the human-friendly name for a field or multiple fields.

set_type($field, $type, array $values = FALSE), set_types(array $types)

Set the type for a field or multiple fields.

set_value($field, $value), set_values(array $values)

Set the allowed values for a field or multiple fields. The value may be an array of items, or it may be a pipe-delimited string which will be converted to a query. The string must be in the following format:

table|index column|value column|where clause|order clause

clear()

Clears all the form fields and associated settings. Use this if you want to re-use a Form object.

get_fields()

Returns an indexed array of arrays with ‘name’ => ‘sanitized value’.
