Features

Flexiblility

You can have your cake and eat it too: enjoy convention and configuration with Core52!

MVC

Core52 uses the familiar Model-View-Controller architecture, with a twist…

Lightweight

Core52 is fast, really fast. We autoload all models and libraries, and lazy load wherever possible.

PHP 5

Core52 leverages the object-oriented greatness that is PHP 5.2*.

Integration

Core52 is pre-integrated with more than a dozen common third-party libraries, including API clients for Twitter and Facebook.

Documentation

We took a cue from EllisLabs and made the documentation for Core52 second to none.

Powerful Form Processing

Hand-coding form validation routines gets old quick–really quick. Core52′s Form library handles validation, rendering, filtering, and sanitizing so you don’t have to.

Secure

Core52 has been tested–and hardened–over more than a year of use, and has been designed to prevent many common security flaws, including XSS, SQL injection, and more.

Error Reporting

Backtraces and query logs are standard in Core52, making bug-squashing a breeze! In production, your application can automatically “phone home” every error report via email–along with GET, POST, and Session dumps.

Raw CLI Power

For those intense behind-the-scenes tasks, you can leverage the full power of Core52 inside commandline scripts with a single include. Multiprocessing? Shared Memory? Daemons? We’ve got you covered there too.

* PHP 5.3 is not yet supported.
